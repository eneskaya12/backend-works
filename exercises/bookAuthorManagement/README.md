# Kitap & Yazar Yönetimi (Basit–Orta)

Amaç: Stream API temel işlemler (filter, map, sorted).

## Sınıflar

Author → id, name

Book → id, title, price, author

## Yapılacak İşlemler

- 100 TL’nin altında olan kitapları listeleyin.

- Yazarlara göre ortalama kitap fiyatlarını Map formatında hesaplayın.

- En pahalı kitabı bulup yazdırın.

- Kitap isimlerini alfabetik olarak sıralayın.

- Adında "Java" geçen kitapları filtreleyin.

- Aynı yazara ait kitapları groupBy ile gruplayın.
